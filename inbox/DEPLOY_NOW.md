3. **账号**：2 个微信账号（A=无年卡，B=无年卡），1 台设备
4. **测试数据**：数据库已预置 4 个专用测试码（见下方）

| 码 | 状态 | 用途 |

| `ZGB-TST1-A3K9` | unused | 正常兑换 C1-C8 |

## 一、用户端 — 核心路径 🔴

| # | 操作 | 预期 |
|:--:|------|------|
| C1 | 账号A → 我的 → 「🎫 使用内测码」 | 进入兑换页，提示「输入 13 位内测码，短横线自动补全」 |
| C2 | 输入 `zgbtst1a3k9`（小写无短横线） | 自动变 `ZGB-TST1-A3K9` |
| C3 | 点击「查询码状态」 | 「该码有效，兑换后将获得 365天基础年卡会员」 |
| C4 | 点击「立即兑换」 | 按钮 loading → 「年卡已激活」+ 到期日（今天+365天） |
| C5 | 选填反馈 → 提交 | 「感谢反馈」 |
| C6 | 「查看会员状态」 | 我的页面显示「基础年卡会员」 |
| C7 | 返回兑换页再次输 `ZGB-TST1-A3K9` | 「你已兑换过该码」 |

## 二、用户端 — 异常码

| # | 操作 | 预期 |
|:--:|------|------|
| E1 | 账号B → `ZGB-TST1-A3K9`（已用） | 「该码已被使用」+ 挽留入口 |
| E2 | 输入 `ABC123` | 前端提示「格式：ZGB-XXXX-XXXX」（红色） |
