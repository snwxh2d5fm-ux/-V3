## BUG-P1-6: 关卡0:抵港前准备被锁

currentPhase: 1,

  '1': { unlocked: true, completed: false }

## BUG-P1-7: 场景速查空白

**文件**: `pages/guidebooks/index/index.js`
**行号**: 199-203
**严重度**: 🟡 — 场景速查Tab始终空白

```js
loadBrowse: function(category) {
  var self = this;
  self.setData({ activeCategory: category });
  var promise = category === '全部' ? cache.fetchAllTasks() : cache.fetchTasks('bySceneTags', { tags: [category] });
  promise.then(function(r) { if (r) self.setData({ browseTasks: r.data || (r.data ? r.data.data : []) }); });
},
```

**根因**: `cache.fetchAllTasks()` 返回 `{ data: cloudFnResp, fromCache, stale }`，其中 `cloudFnResp` 是云函数返回值。`r.data` 是 object 不是数组，`r.data || (r.data ? r.data.data : [])` 逻辑始终取 `r.data`（truthy object），但 `browseTasks` 需要的是数组。

**修复**: `r.data.data || r.data.tasks || []` — 从响应中正确解包任务列表。
