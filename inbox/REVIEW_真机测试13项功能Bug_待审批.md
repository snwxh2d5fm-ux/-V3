## 🔴 P0 — 阻塞上线

### #4 提醒器未按优才项目进度生成时间线
- **模块**: reminders
- **问题**: 提醒器没有按照优才管理项目进度和周期生成时间线和提醒
- **预期**: 选择优才路径后 → 自动生成节点时间线（提交→补件→获批→激活→续签→永居），每个节点带提醒
- **影响**: 核心功能完全失效

### #5 用户贡献攻略只有标题无内容
- **模块**: guidebooks
- **问题**: 用户提交的攻略内容空白，只显示标题
- **影响**: UGC 模块功能阻断，信任度归零

### #7 切换配偶后仍可见本人证件
- **模块**: documents
- **问题**: 切换到「配偶」所属人后，仍能看到本人的身份证证件
- **预期**: identityOwner 切换 = 完全数据隔离，self/spouse/child 独立
- **影响**: **数据隔离失效，隐私泄露风险**

### #10 智能材料清单路径数据未同步
- **模块**: documents / 材料清单
- **问题**: 智能材料清单中，选择路径后数据未同步到清单
- **影响**: 材料清单功能阻断

### #11 各证件线框全部错误
- **模块**: documents / OCR
- **问题**: 所有证件的线框都不对，需要从脱敏数据库中找真实案例做参照
- **影响**: OCR 定位全部失败，证件识别核心链路断裂

### #12 脱敏未按用户证件进行遮挡
- **模块**: documents / K2安全
- **问题**: 脱敏没有根据用户的证件类型进行对应遮挡
- **影响**: K2 安全合规红线

## 🟡 P1 — 体验/标准违规

### #1 流程控长周期路径提醒需去掉
- **模块**: process
- **问题**: 长周期路径（如优才8年）提醒项冗余，用户困惑
- **方案**: 去掉长周期路径的自动提醒，或改为可选

### #2 证件夹头部颜色不统一
- **模块**: documents / UI
- **问题**: 证件夹页面头部颜色与整体 UI 标准不一致

### #3 证件夹 & 优才计划材料准备颜色不统一
- **模块**: documents / UI
- **问题**: 「证件夹」和「优才计划.材料准备」两个页面的颜色未按 UI 标准统一

### #8 证件图片缺少旋转/缩放/裁剪功能
- **模块**: documents
- **问题**: 旋转不保存；没有放大、缩小、裁截对线框的功能
- **预期**: 证件拍照后支持旋转(保存角度)+ 双指缩放 + 裁剪框对齐

### #9 证件添加流程太长
- **模块**: documents
- **问题**: 证件添加环节太多，最后确认环节需要去掉或前移
- **方案**: 精简步骤，确认步骤前移至预览页

### #13 预检详情%空白
- **模块**: precheck
- **问题**: 预检详情页面的百分比显示为空白
- **预期**: 显示具体预检通过率百分比
