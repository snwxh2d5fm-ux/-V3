# 住港伴 V3 — Claude Code 配置

## 项目概述

住港伴（HK Companion）是一款微信小程序，帮助用户管理香港签证申请流程、证件文档、时间线和提醒。

## 技术栈

- 微信小程序原生框架
- CloudBase 云开发（云函数、云数据库、云存储）
- Jest 单元测试 + E2E 测试

## 目录结构

- `pages/` — 小程序页面
- `cloudfunctions/` — 云函数
- `data/` — 模板数据与常量
- `utils/` — 工具函数
- `tests/` — E2E 测试
- `__tests__/` — 单元测试

## 开发约定

- 提交信息使用中文摘要，格式：`模块: 变更说明`
- 云函数使用 Node.js 18 运行时
- 测试运行: `npx jest`
