# 住港伴 V4 更新日志

> 发布日期：2026-05-21 | 基于 V3 已发正式版本

---

## 🆕 新增功能

### 双闸门决策保护

未登录或未确认身份状态的用户，在尝试选择身份规划路径时将被引导完成登录和身份确认，防止匿名用户创建流程线产生脏数据。

- **5 个入口全面保护**：流程控直接选路径、模板流程创建、独立路径选择页、AI 评估结果选路径、AI 对话推荐采纳路径
- **智能引导**：未登录用户弹出登录半屏，已登录但跳过身份的用户弹出身份确认半屏
- **支付安全**：攻略书购买解锁和会员订阅增加登录门禁

### 路径卡片交互优化

流程控的路径选择卡片支持点击展开查看详情（无需登录），展开后点击「选择此路径」按钮才触发闸门检查。访客可以自由浏览路径信息再做决策。

### 提醒器路径切换封存

切换身份规划路径时，旧路径的提醒自动封存，不再与新路径提醒混合显示。切回原路径时自动解封恢复。

## 🔧 修复

- 修复 `__active_process_id__` 存储为路径类型字符串而非唯一流程 ID 的数据完整性问题
- 修复 gate-sheet 登录流程中 `phoneBound`/`membershipLevel` 未写入 globalData 的状态丢失
- 修复 `VALID_STATUSES` 白名单未生效的闸门校验缺陷
- 修复流程控 `onSelectDirectPath` 缺少双击防护可能创建重复流程线
- 修复未登录用户可通过攻略书直接拉起微信支付的漏洞
- 修复 path-select 页面路径覆写顺序导致旧提醒无法封存的问题
