## 1. 问题定义

### 1.1 触发事件

2026年5月25日，V4.2审核版本真机测试中，流程控页面点击路径选择时触发 `_.track is not a function` 崩溃，根因为 `tracker.track()` 调用了tracker模块中不存在的API（正确API为 `tracker.event()`）。此缺陷自V4.2发版（commit `4aea8c9`）起已存在于代码中，因从未触发"新建流程"路径而未被发现。

### 1.2 缺陷画像

全项目扫描发现同类别预存缺陷4项：

| 缺陷 | 位置 | 触发场景 | 用户影响 |
|------|------|----------|----------|
| tracker API错误 | process/index.js, assessment-result/index.js (3处) | 新建流程 / 自评选路径 | 流程创建崩溃，无法使用核心功能 |
| image-process路径错误 | documents-add/index.js | 文档图片裁剪确认 | 证件裁剪功能崩溃 |
| guidebook-data 文件缺失 | guidebooks-detail/index.js | 访问废弃攻略详情页 | 已标记 @deprecated，但若路由可达则白屏 |

### 1.3 共同特征

```
这类缺陷不被常规测试捕获的原因：

代码写入了   →   但执行路径从未触发
Lint不报错    →   语法正确、引用合法
Jest不覆盖   →   测试不走到这个分支
真机不漏     →   测试场景未覆盖特定操作
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
只有在特定用户操作下才会崩溃
```

## 2. 影响评估

### 2.1 用户影响矩阵

| 场景 | 影响用户 | 触发概率 | 严重度 | 优先级 |
|------|----------|:--:|:--:|:--:|
| 新用户创建流程 → 选路径 | 全部新用户 | 100% | 🔴 核心功能不可用 | P0 |
| 自评完成 → 选推荐路径 | 使用自评的用户 | 100% | 🔴 选路径环节崩溃 | P0 |
| 证件上传 → 裁剪确认 | 需要裁剪证件的用户 | 100% | 🔴 上传功能不可用 | P0 |
| 访问废弃攻略详情页 | 极少（已废弃） | 低 | 🟡 白屏 | P2 |

### 2.2 业务影响

- **新用户完整体验路径断裂**：新用户从注册 → 选身份 → 选路径的三步流程中，第三步崩溃
- **自评转化归零**：用户完成自评后无法采纳推荐的路径方案，整个自评功能失去转化意义
- **证件管理降级**：无法裁剪证件图片，用户只能上传原图，影响OCR准确率
- **审核风险**：微信审核员测试完整流程时可能触发崩溃，导致审核不通过

### 2.3 与审核拒绝的关系

本次V4.2审核被拒三项均为**交互合规问题**（隐私同意、登录后置、AI标识），与预存缺陷无关。但预存缺陷若不修复，在审核员测试或真机用户使用中都将暴露，构成 **隐性审核风险**。

## 4. 需求详情

### REQ-1: 流程控路径选择修复

**用户故事**：作为新用户，我选择身份状态后，希望在流程控页面点选具体身份路径，系统应跳转到路径选择页而非崩溃。

**验收标准**：
- [ ] 新用户 → 状态选择页 → 流程控Tab → 点击路径卡片 → `selectTemplate` 正常执行
- [ ] `tracker.event('process_created', {...})` 被正常调用，不再抛出 `is not a function`
- [ ] 路径选择面板正常弹出，可点击具体路径
- [ ] 选择路径后流程创建成功，阶段指示器正常展示

### REQ-2: 自评结果页路径选择修复

**用户故事**：作为完成自评的用户，我点击AI推荐的路径方案后，系统应完成路径选择而非崩溃。

**验收标准**：
- [ ] 自评 → 查看结果 → 点击推荐路径 → `onGatePassed` 正常执行
- [ ] `tracker.event('assessment_completed', ...)` 正常
- [ ] `tracker.event('path_selected', ...)` 正常
- [ ] 路径选择后正确跳转到流程控Tab

### REQ-3: 证件裁剪功能修复

**用户故事**：作为上传证件的用户，我在证件预览页点击裁剪确认后，系统应完成裁剪处理而非白屏。

**验收标准**：
- [ ] 证件添加页 → 拍照/选图 → 预览 → 点击裁剪确认 → `confirmCrop()` 正常执行
- [ ] `require('../../utils/image-process')` 成功加载模块
- [ ] 裁剪后的图片正确显示

### REQ-4: 废弃攻略详情页清理

**用户故事**：无用户故事。此页面已标记 `@deprecated 2026-05-14`，当前不可达。

**验收标准**：
- [ ] `subpkg-guide/pages/guidebooks-detail/` 目录及所有文件已删除
- [ ] `app.json` 中对应页面注册已移除
- [ ] 编译无报错
- [ ] 无其他页面引用此路径
