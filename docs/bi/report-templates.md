## 四、报表生成自动化

### 4.1 日报 (Cron: 每天 09:00)

触发 `usage-tracker` 的 stats 查询 + WE分析 Dashboard

### 4.2 周报 (Cron: 每周一 08:00)

BI Agent 手动触发流程:
1. 查询 usage-tracker stats (path_preference + funnel + user_summary)
2. 查询 orders 本周收入
3. 查询 policy_updates 本周政策
4. 生成周报 Markdown → 推送通知

### 4.3 月报 (Cron: 每月1日 08:00)

BI Agent 手动触发流程:
1. 查询所有月聚合指标
2. 市场竞品扫描
3. 留存矩阵计算
4. 生成月报 Markdown → 推送通知
