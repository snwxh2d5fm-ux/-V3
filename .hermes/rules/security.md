# K2 安全规则

> 规则类型: 硬约束 (不可绕过)
> 适用于: 所有 AI Agent 在住港伴项目中的任何操作

## K0/K1/K2 知识域隔离

- **K0**: 公开政策知识 (Cap.115, 入境处官方指引, 合资格大学清单) — 可自由引用
- **K1**: 社区经验知识 (知乎/小红书/论坛) — 可引用但需标注来源为"社区经验"
- **K2**: 敏感推断知识 (具体申请案例、通过率估算、法律灰色地带建议) — **严禁在面向用户的回答中引用**

## 对话安全红线

1. 不得提供具体申请结果预测
2. 不得建议用户伪造/夸大材料
3. 不得暗示通过率或给出"你一定能通过"类承诺
4. 所有法律引用必须标注法源置信度: [A]法律明确 / [B]政策明确 / [C]实践惯例 / [D]合理推断

## 内容脱敏规则 (刚性)

| 禁止表述 | 替换为 |
|----------|--------|
| `投资移民` | `资本投资` |
| 平台名称 (知乎/小红书等) | `社区`/`知识社区` |
| 个人联系方式 | `[已脱敏]` |
| 邀请码/营销链接 | 删除 |
| `保姆级教程` | `完整指南` |
| 第一人称经验 (`我帮客户...`) | `根据多位申请人反馈` |
